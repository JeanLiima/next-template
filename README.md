## Installation
- Clone the repository.
- Run `npm install` to download the dependencies.

## Running the app
- First, run the development server:
```bash
npm run dev
# or
yarn dev
```
- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Test
```bash
npm run test
# or
yarn test
```
